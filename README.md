# Indy Voter Compass

## Overview
Indy Voter Compass is a tool designed to help independent voters navigate political landscapes and make informed voting decisions.

## Features
- Voter preference analysis
- Candidate comparison tools
- Issue-based filtering
- Non-partisan information resources

## Getting Started
1. Clone the repository
2. Install dependencies
3. Configure local environment
4. Run the application
